# Exponeciação

$$
a^n = a \cdot a \cdot a \cdots a \quad (n \text{ vezes})
$$

- (a) = base
- (n) = expoente

## Casos básicos

- Indefinição: $0^0$ é indefinido
- Base $0$: $0^n = 0 \quad (n > 0)$
- Base $1$: $1^n = 1$
- Expoente $1$: $a^1 = a$
- Expoente $0$: $a^0 = 1 \quad (a \neq 0)$
- Expoente $>0$: $2^3 = 2 \cdot 2 \cdot 2 = 8$
- Expoente $<0$: $a^{-n} = \frac{1}{a^n}$
  - Ex: $2^{-3} = \frac{1}{2^3} = \frac{1}{8}$

## Propriedades

- Produto de potências (mesma base): $a^m \cdot a^n = a^{m+n}$
  - Ex: $2^3 \cdot 2^2 = 2^5 = 32$
- Divisão de potências: $\frac{a^m}{a^n} = a^{m-n}$
  - Ex: $\frac{2^5}{2^2} = 2^3 = 8$
- Potência de potência: $(a^m)^n  = (a^n)^m = a^{m \cdot n}$
  - Ex: $(2^3)^2 = 2^6 = 64$
- Potência de produto: $(ab)^n = a^n \cdot b^n$
  - Ex: $(2 \cdot 3)^2 = 2^2 \cdot 3^2 = 36$
- Potência de quociente: $\left(\frac{a}{b}\right)^n = \frac{a^n}{b^n}$
- Expoente fracionário (raiz): $a^{\frac{m}{n}} = \sqrt[n]{a^m}$
  - $a^{1/2} = \sqrt{a}$
  - $a^{1/3} = \sqrt[3]{a}$
  - Ex: $8^{1/3} = 2$
- Expoente negativo + fração: $a^{-m/n} = \frac{1}{\sqrt[n]{a^m}}$
- Produto com expoentes iguais: $a^n \cdot b^n = (ab)^n$
- Comparação de potências:
  - Mesma base: $a^x = a^y \Rightarrow x = y \quad (a>0, a\neq1)$ (compare expoentes)
    - Ex: $2^5 > 2^3$
  - Mesmo expoente: $a^x = b^x \Rightarrow a = b$ (compare bases)
    - Ex: $5^2 > 3^2$

## Base negativa

- Com parênteses: $(-2)^2 = 4$
- Sem parênteses: $-2^2 = -4$

**ordem importa!**

## Crescimento exponencial

Função: $f(x) = a^x$

- (a > 1) → cresce 📈
- (0 < a < 1) → decresce 📉

## Transformações úteis

- Igualar bases: $8 = 2^3$ ou  $16 = 2^4$
  - Exemplo: $2^x = 8$
  - logo $2^x = 2^3 \Rightarrow x = 3$

## Erros comuns

- ❌ (a^m + a^n = a^{m+n}) → ERRADO
- ✔ só vale para multiplicação
- ❌ ((a+b)^n = a^n + b^n) → ERRADO

## Casos especiais

- Indefinição: $0^0 \text{ é indefinido}$
- Notação científica: $a \times 10^n$
  - Ex: $300000 = 3 \times 10^5$

## Ligação com logaritmo

- Exponencial: $a^x = b$
- Log: $x = \log_a b$

## Estratégia para resolver exercícios

1. Igualar bases
2. Aplicar propriedades
3. Simplificar expoentes
4. Resolver equação

## RESUMO FINAL

- Soma expoentes → multiplicação
- Subtrai expoentes → divisão
- Multiplica expoentes → potência de potência
- Fração → raiz
- Negativo → inverso
