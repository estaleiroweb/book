# Fatoração Direta de Polinomios

Ex:
$$
x^3 - x^2 - x + 1 = 0
$$

Agrupando:
$$
(x^3 - x^2) - (x - 1) = 0
\\
x^2(x - 1) - 1(x - 1) = 0
\\
(x - 1)(x^2 - 1) = 0
\\
(x - 1)^2(x + 1) = 0
$$

logo

para $(x_1,x_2)$:
$$
(x - 1)^2=\frac{0}{(x + 1)}\\
(x - 1)^2=0\\
x^2 - 2x + 1=0\\
\Delta = b^2 - 4ac = 4-4=0\\
x = \frac{-b \pm \sqrt{\Delta}}{2a} = \frac{2}{2}\\
x = 1
$$

para $(x_3)$:
$$
(x + 1) = \frac{0}{(x - 1)^2}\\
x+1=0\\
x=-1
$$

Raízes:

- $x = 1$ (dupla)
- $x = -1$
