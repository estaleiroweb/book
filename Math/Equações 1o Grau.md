# Equações do 1º Grau

$$
f(x) = ax + b \quad (a \neq 0)
$$

- (a): coeficiente angular (inclinação da reta)
- (b): coeficiente linear (onde corta o eixo Y ou “altura inicial”)

O gráfico é sempre uma **reta**

## Raiz

- Onde o gráfico cruza o eixo X
- Solução da equação $f(x)=0$

$$
ax + b = 0 \\
logo\\
x = -\frac{b}{a}
$$

- Sempre existe **uma única solução**

## Coeficiente angular (inclinação)

$$
a = \frac{\Delta y}{\Delta x}
$$

- Interpretação:
  - $a > 0$: reta sobe ↗ (Crescente)

    ![a>0](https://s1.static.brasilescola.uol.com.br/be/e/6(12).jpg)
  - $a < 0$: reta desce ↘ (Decrescente)

    ![a<0](https://s1.static.brasilescola.uol.com.br/be/e/7(9).jpg)
  - $a = 0$: reta horizontal (Constante)

    ![a=0](https://static.preparaenem.com/2022/12/grafico-funcao-afim-constante.jpg)]

## Forma ponto-inclinação

Se você conhece um ponto ($x_0, y_0$):

$$
y - y_0 = a(x - x_0)
$$

## Forma por dois pontos

Se tem dois pontos:

$$
a = \frac{y_2 - y_1}{x_2 - x_1}
$$

Depois monta:
$$
y = ax + b
$$

## Forma geral da equação (ou implícita)

$$
ax + by + c = 0
$$

Pode ser transformada para:
$$
y = mx + n
$$

## Particularidades

- **Interseções**:
  - **Eixo Y:** $f(0) = b$
  - **Eixo X:** $x = -\frac{b}{a}$
- **Paralelismo**: se $a_1 = a_2$ Duas retas são paralelas
- **Perpendicularidade**: Duas retas são perpendiculares se ($a_1 \cdot a_2 = -1$)
- **Imagem da função**: $\text{Imagem} = \mathbb{R}$ A reta vai para infinito positivo e negativo
- **Domínio**: $\text{Domínio} = \mathbb{R}$
- Constante: Se $a = 0$ logo $f(x) = b$
  - Reta horizontal
  - Não corta o eixo X (se (b \neq 0))
- Exemplo: $f(x) = 2x + 4$
  - Raiz: $2x + 4 = 0 \Rightarrow x = -2$
  - Interseção Y: $f(0) = 4$
  - Inclinação: sobe 2 unidades a cada 1 no eixo X

## Comparação rápida com 2º grau

| Característica | 1º grau    | 2º grau   |
| -------------- | ---------- | --------- |
| Gráfico        | Reta       | Parábola  |
| Raízes         | 1          | 0, 1 ou 2 |
| Curvatura      | Não        | Sim       |
| Δ (delta)      | Não existe | Existe    |
