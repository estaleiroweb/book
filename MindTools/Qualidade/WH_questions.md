# WH Questions

Tendo em vista a grande utilização das perguntas para investigar vários tipos de assuntos, outras perguntas podem ser feitas, o que gerou acrônimos como 5W2H, 6WH e outras variações.
Uma lista bastante completa de perguntas adicionais é a seguinte:

- What? (O quê?)
- Why? (Por quê?)
- Where? (Onde?)
- When? (Quando?)
- Who? (Quem?)
- How Much? (Quanto?)
- How? (Como?)
- Wins? (Condições de sucesso?)
- Whom? (Quem? como objeto)
- Whose? (De quem? como possuidor)
- What kind of? (Que tipos?)
- Which? (Quais? entre uma lista)
- How many? (Quantos?)
- How far? (Qual a distância?)
- How long? (Por quanto tempo?)
- How often? (Com que frequência?)
