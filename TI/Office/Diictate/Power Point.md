# Power Point Dictate

No PowerPoint, a função de "Dictate" (Ditado) também permite o uso de comandos de voz em português para facilitar a criação e edição de apresentações. Aqui estão alguns comandos úteis:

## Comandos para Navegação:

- Nova linha: Move o cursor para uma nova linha na mesma caixa de texto.
- Novo parágrafo: Insere um novo parágrafo na mesma caixa de texto.
- Próximo slide: Move para o próximo slide.
- Slide anterior: Volta para o slide anterior.
- Comandos para Formatação:
- Negrito: Aplica negrito ao texto selecionado.
- Itálico: Aplica itálico ao texto selecionado.
- Sublinhado: Aplica sublinhado ao texto selecionado.
- Centralizar texto: Centraliza o texto na caixa de texto.
- Alinhar à esquerda: Alinha o texto à esquerda.
- Alinhar à direita: Alinha o texto à direita.
- Tamanho da fonte [número]: Ajusta o tamanho da fonte (por exemplo, "Tamanho da fonte 20").
- Comandos para Inserção:
- Inserir nova caixa de texto: Insere uma nova caixa de texto no slide.
- Inserir imagem: Abre o diálogo para inserir uma imagem.
- Inserir tabela: Abre o diálogo para inserir uma tabela.
- Inserir gráfico: Abre o diálogo para inserir um gráfico.

## Comandos Especiais:

- Inserir símbolo [nome do símbolo]: Insere um símbolo específico (como "arroba" ou "asterisco").
- Selecionar tudo: Seleciona todo o texto na caixa de texto ativa.
- Copiar: Copia o texto ou objeto selecionado.
- Colar: Cola o conteúdo copiado.
- Salvar apresentação: Salva a apresentação atual.
- Abrir apresentação: Abre uma apresentação existente.

## Comandos de Pontuação e Símbolos:

- Inserir vírgula: Insere uma vírgula.
- Inserir ponto final: Insere um ponto final.
- Inserir dois pontos: Insere dois pontos.
- Abrir aspas: Insere uma aspas de abertura.
- Fechar aspas: Insere uma aspas de fechamento.
