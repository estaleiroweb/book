# Setas

| Símbolo           | Comando LaTeX     | Descrição             | Char |
| ----------------- | ----------------- | --------------------- | ---- |
| $\rightarrow$     | `\rightarrow`     | Seta para a direita   | →    |
| $\leftarrow$      | `\leftarrow`      | Seta para a esquerda  | ←    |
| $\uparrow$        | `\uparrow`        | Seta para cima        | ↑    |
| $\downarrow$      | `\downarrow`      | Seta para baixo       | ↓    |
| $\leftrightarrow$ | `\leftrightarrow` | Seta dupla horizontal | ↔    |
