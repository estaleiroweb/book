# Operadores

| Categoria     | Símbolo     | Comando LaTeX | Descrição      | Char |
| ------------- | ----------- | ------------- | -------------- | ---- |
| **Direct**    | $:$         | `:`           |                | :    |
|               | $!$         | `!`           |                | !    |
|               | $,$         | `,`           |                | ,    |
|               | $;$         | `;`           |                | ;    |
|               | $.$         | `.`           |                | .    |
|               | $'$         | `'`           |                | '    |
|               | $"$         | `"`           |                | "    |
| **Núméricos** | $+$         | `+`           | Adição         | +    |
|               | $-$         | `-`           | Subtração      | -    |
|               | $*$         | `*`           | Multiplicação  | *    |
|               | $/$         | `/`           | Divisão        | /    |
|               | $\div$      | \div          | ÷              | ÷    |
|               | $\bmod$     | \bmod         | mod            |      |
|               | $\times$    | \times        | ×              | ×    |
|               | $\pm$       | \pm           | ±              | ±    |
|               | $\star$     | \star         | ⋆              | ⋆    |
|               | $\setminus$ | \setminus     | ∖              | ∖    |
| **Lógicos**   | $=$         | `=`           | Igualdade      | =    |
|               | $\equiv$    | \equiv        | ≡              | ≡    |
|               | $\neq$      | `\neq`        | Diferente      |      |
|               | $<$         | `<`           | Menor          | <    |
|               | $>$         | `>`           | Maior          | >    |
|               | $\lt$       | `\lt`         | Menor          | <    |
|               | $\gt$       | `\gt`         | Maior          | >    |
|               | $\leq$      | `\leq`        | Menor ou igual | ≤    |
|               | $\geq$      | `\geq`        | Maior ou igual | ≥    |
