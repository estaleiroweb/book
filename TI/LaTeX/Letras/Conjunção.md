# Conjunção

| Símbolo        | Comando LaTeX  | Descrição                           |
| -------------- | -------------- | ----------------------------------- |
| $\tilde{a}$    | `\tilde{a}`    | Til em cima de `a`                  |
| $\bar{a}$      | `\bar{a}`      | Barra em cima de `a`                |
| $\hat{a}$      | `\hat{a}`      | Circunflexo em cima de `a`          |
| $\check{a}$    | `\check{a}`    | Sinal de verificação em cima de `a` |
| $\breve{a}$    | `\breve{a}`    | Breve em cima de `a`                |
| $\dot{a}$      | `\dot{a}`      | Ponto em cima de `a`                |
| $\ddot{a}$     | `\ddot{a}`     | Dois pontos em cima de `a`          |
| $\vec{a}$      | `\vec{a}`      | Setinha sobre `a`                   |
| $\grave{a}$    | `\grave{a}`    | Acento Grave                        |
| $\acute{a}$    | `\acute{a}`    | Acento  Agudo                       |
| $\not{a}$      | `\not{a}`      | Corte                               |
| $\mathring{a}$ | `\mathring{a}` |                                     |
