# Caracteres de escape

Para exibir um caractere literal que, de outra forma, seria usado para formatar texto em um documento Markdown, adicione uma barra invertida ( \) na frente do caractere.

\* Without the backslash, this would be a bullet in an unordered list.
A saída renderizada se parece com isso:

* Sem a barra invertida, isso seria um marcador em uma lista não ordenada.

## Personagens que você pode escapar

Você pode usar uma barra invertida para escapar os seguintes caracteres.

| Personagem | Nome                                                           |
| ---------- | -------------------------------------------------------------- |
| \\         | barra invertida                                                |
| `          | acento grave (veja também [escape de acentos graves no código](https://www.markdownguide.org/basic-syntax/#escaping-backticks) ) |
| *          | asterisco                                                      |
| _          | sublinhado                                                     |
| { }        | chaves                                                         |
| [ ]        | colchetes                                                      |
| < >        | colchetes angulares                                            |
| ( )        | parênteses                                                     |
| #          | sinal de libra                                                 |
| +          | sinal de mais                                                  |
| -          | sinal de menos (hífen)                                         |
| .          | ponto                                                          |
| !          | ponto de exclamação                                            |
| \|         | pipe (veja também [pipe de escape em tabelas](https://www.markdownguide.org/extended-syntax/#escaping-pipe-characters-in-tables) )                  |
