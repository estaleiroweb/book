# Ambiente de Trabalho

Neste tema, iremos tratar de configurações do nosso ambiente de trabalho para desenvolvimento de softwares.

O objetivo desse tema é transformar sua mesa de trabalho em um espaço facilitador para seu trabalho aumentando sua produtividade.

A padronização do seu ambiente de trabalho lhe permite usar vários aplicativos de forma similar inclusive as hot-keys.

Destacaremos:
- [PUTTY](putty.md): Conector para terminal
- [XShell](xshell.md): Conector para terminal
- [WinSCP](winscp.md): SFTP Explorer
- [Tunnel](tunnel.md): Conexão direta a um serviço
- [Teclado](keyboard.md): Conheça seu teclado e use todo
- [git/GiHub/GitLab](git.md): Controle de Versão de Arquivos
- [Git Bash](gitbash.md): App Windows para controle de versão
- [VSCode](vscode.md): Suite de desenvolvimento e documentação

Mais tarde, em outro bloco futuro, falaremos sobre de os itens abaixo e a integração com os itens acima:
- Documentação
  - Markdown: https://www.markdownguide.org/
  - Mermaid: https://mermaid.js.org/ecosystem/tutorials.html
  - LaTex: https://www.latex-project.org/help/documentation/
- Container
  - Docker Desktop: https://www.docker.com/products/docker-desktop/
- Programação
  - PHP: https://www.php.net/
  - Python: https://www.python.org/
