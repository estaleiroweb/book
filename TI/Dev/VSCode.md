## Hot keys

| Hot Key              | Action                                         |
| -------------------- | ---------------------------------------------- |
| F2                   | Rename filename                                |
| F2 F2                | Rename full filename                           |
| F2 F2 F2             | Rename extention name                          |
| Crtl+B               | Hide/Show Explorator                           |
| Crtl+K Z             | Enter/Go Out full screen                       |
| Ctrl+P               | List of Files                                  |
| Ctrl+Shift+P         | Bar command                                    |
| Ctrl+Shift+P compare | Comparar arquivos                              |
| ALT+F5               | Next change in compare                         |
| SHIFT+ALT+F5         | Preview change in compare                      |
| Crtl+K C             | Compare with clipboard                         |
| Ctrl+P @             | List of Class/Functions                        |
| Ctrl+P @:            | List of Class/Functions Ordered                |
| Ctrl+Shift+.         | Nav in breacrumbs                              |
| Ctrl+.               | Em cima do erro. sugere acertos                |
| Crtl+Shift+L         | Seleção de todas as ocorrências para alteração |
| Ctrl+;               | Comment line                                   |
| Ctrl+Shift+;         | Comment block (Ctrl+Shift+A)                   |
| Crtl+Alt+F           | Formatar Documento                             |
