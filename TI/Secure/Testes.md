

Aqui estão alguns sites úteis para testar certificados HTTPS e DNS de outro site:

### **Testes de Certificado HTTPS**
1. **SSL Labs – SSL Server Test**  
   [Qualys SSL Labs](https://www.ssllabs.com/ssltest/)  
   Verifica a qualidade e segurança da implementação SSL/TLS de um site, incluindo a validade do certificado, cadeia de certificação, protocolos suportados e possíveis vulnerabilidades.

2. **SSL Checker**  
   [SSL Checker](https://www.sslshopper.com/ssl-checker.html)  
   Permite verificar rapidamente se um certificado é válido e se está corretamente instalado.

3. **Hardenize**  
   [Hardenize](https://www.hardenize.com/)  
   Realiza uma análise completa, incluindo HTTPS, HSTS, segurança de cabeçalhos e muito mais.

---

### **Testes de DNS**
1. **DNS Checker**  
   [DNS Checker](https://dnschecker.org/)  
   Permite verificar a propagação DNS globalmente, mostrando os resultados de diferentes servidores DNS ao redor do mundo.

2. **WhatsMyDNS**  
   [WhatsMyDNS](https://www.whatsmydns.net/)  
   Testa a resolução DNS em vários locais do mundo, útil para checar se mudanças propagaram corretamente.

3. **MXToolbox – DNS Check**  
   [MXToolbox DNS Check](https://mxtoolbox.com/DNSLookup.aspx)  
   Realiza consultas DNS detalhadas para registros A, CNAME, MX, TXT, etc.

4. **IntoDNS**  
   [IntoDNS](https://intodns.com/)  
   Verifica a configuração DNS de um domínio, identificando problemas comuns de configuração.

5. **DNSViz**  
   [DNSViz](http://dnsviz.net/)  
   Gera uma visualização detalhada da resolução DNS de um domínio, útil para analisar problemas de DNSSEC.

---
